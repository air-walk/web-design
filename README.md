# web-design

1. **border-boxing:**
```css
* {
    -webkit-box-sizing: border-box;
       -moz-box-sizing: border-box;
        -ms-box-sizing: border-box;
            box-sizing: border-box;
}
```
2. **flex-boxing:**
```css
.parent-container {
  display:   -webkit-flex;
  display:   flex;
  flex-wrap: wrap;
}
```
